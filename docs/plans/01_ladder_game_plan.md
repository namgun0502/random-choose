# 01. 사다리 게임 탭 추가 구현 계획서

대포 랜덤 당첨 게임과 연동되는 "사다리 게임" 기능을 신규 탭으로 추가하여 멀티 게임 플랫폼 형태로 확장합니다.

---

## 📌 주요 요구사항

1. **상단 탭(Tab) 메뉴 제공**
   - [💣 대포 게임] 탭과 [🪜 사다리 게임] 탭을 상단에 배치하여 자유롭게 오갈 수 있도록 합니다.
   - 두 게임은 동일한 **참여자 목록**을 공유(연동)합니다.

2. **사다리 게임 구조 및 설정**
   - **상단**: 참여자 이름들이 가로로 정렬되어 표시됩니다. (현재 등록된 참여자들과 실시간 연동)
   - **하단**: 사용자가 직접 입력한 **상품(벌칙/선물 등)**들이 매칭되도록 가로로 정렬됩니다.
   - **상품 등록 영역**: 참여자 수와 동일한 개수의 상품 입력칸을 제공하고, 사용자가 직접 텍스트를 입력할 수 있게 합니다.

3. **사다리 타기 애니메이션 & 진행**
   - HTML5 Canvas를 활용하여 사다리 선(세로선 및 랜덤 가로선)을 그립니다.
   - **[시작]** 버튼을 누르면 위에서부터 각각의 이름이 사다리 선을 타고 내려가 아래의 상품에 매칭되는 과정을 애니메이션 효과와 함께 보여줍니다.

## 🧪 검증 계획

- **기능 테스트**: 대포 게임에서 이름을 등록/삭제하고 사다리 게임 탭으로 이동 시 실시간으로 참여자 이름과 수량이 맞춰지는지 검증.
- **예외 처리**: 참여자가 2명 미만일 때 게임 시작이 불가능하도록 예외 처리.
- **상품 매칭**: 입력한 상품들이 사다리 타고 내려간 사람과 1:1로 겹치지 않고 정확히 매칭되는지 확인.
